# Drift Guard

## Purpose

This file protects the project from uncontrolled expansion.

## Current Build Target

Slice 1: Basic iron production with typed ports, dirty water pressure, diagnostics, graph repair, and objective completion.

## Do Not Add Yet

- full 36-material catalog;
- gases;
- advanced chemistry;
- Closed-Line Seed;
- Closed-Line Certification;
- route medals;
- large milestone tree;
- automation/control system;
- multiple scenario modifiers;
- desktop packaging;
- multiplayer;
- modding;
- save migration system beyond simple prototype needs;
- final art pipeline;
- sound system beyond simple feedback if needed.

## Approved Expansion Rule

A new feature may be added only if it directly supports Slice 1 or is explicitly approved by the director.

## Parking Lot Rule

Good ideas go to `docs/11_Parking_Lot.md`.

They do not enter implementation automatically.
