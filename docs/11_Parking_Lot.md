# Parking Lot

Use this file for good ideas that are not approved for current implementation.

## Format

```markdown
## Idea Title

### Summary

### Why It Might Be Good

### Why It Is Not Now

### Revisit Trigger
```

## Ideas

## Desktop Shell Decision

### Summary

Choose between browser-first, Electron, CEF + React + C++, Tauri + React, Godot, or another shell.

### Why It Might Be Good

The final game likely needs a desktop delivery strategy.

### Why It Is Not Now

Slice 1 playability is the immediate priority. Premature stack migration could slow development.

### Revisit Trigger

Revisit after Slice 1 is playable or if the current React Flow path blocks Slice 1 acceptance.

## Closed-Line Certification Full System

### Summary

Implement final certification metrics, Closed-Line Seed, functional victory slots, source cutoff, and route medals.

### Why It Might Be Good

This is the long-term endgame identity.

### Why It Is Not Now

Slice 1 should only prove the small graph loop.

### Revisit Trigger

Revisit after core production, diagnostics, upgrades, and recovery loops are proven.
